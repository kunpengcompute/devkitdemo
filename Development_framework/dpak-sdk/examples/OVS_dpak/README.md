# ovs卸载 demo

简体中文 | [English](README_en.md)

## 介绍
ovs卸载 demo是参考实现

此demo主要用于为用户签发X509证书的场景

## 使用教程

1. 获取代码

   ```shell
   git clone https://github.com/kunpengcompute/devkitdemo.git
   ```

2. 进入到项目根路径

   ```shell
   cd ./devkitdemo/Development_framework/dpak-sdk/examples/OVS_dpak/script
   ```
3. 运行demo

   ```shell
   bash run_demo.sh
   ```
